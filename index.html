Itchy Feet

Full Stack Web App | 1 week pair project | General Assembly.

JavaScript, React, Python, Flask & SQL

This is a fully mobile responsive CRUD application we built using Python, and a pre-populated data package to create holiday destinations that can be created by users, ‘liked’ and saved to a trip wishlist page. The backend was set up using Python, Flask and Sql, for the frontend we used React and JavaScript, styled using SASS. We used a Mapbox API and a skyscanner widget for extra features. Together we set up the backend models, individually I created the homepage and worked on the like feature, as well as populating the ‘Wishlist’ and ‘My Trips’ pages with filtered information.

Munch

Full Stack Web App | 10 day group project | General Assembly.

JavaScript, Express, React, Node.js

We created an online portfolio which allowed users to create a food-focused portfolio to store recipes and restaurant reviews. Using RESTful API, MongoDB, Express, Node.js. As a pair, I worked on the backend models, using Mongoose, and controllers and individually I created the newsfeed that filtered data specific to the users chosen categories, as well as making the ‘like’ function which was a challenge. We styled using a combination of Bulma and SASS. I carried out full testing for one CRUD path using Mocha and Chai.

The Sorting Hat

48 hour pair project | General Assembly.

Individually I worked on getting the information for each house and the displaying of the houses. Styled with CSS.

Monkey Invaders

1 week individual project | General Assembly.

JavaScript, HTML, CSS

General Assembly. Individual challenge. 1 week. I built a front-end interactive Space-Invaders style game using HTML, CSS and JavaScript. The game included keyboard movement, a score a high-score stored in local storage and six levels with increasingly fast moving monkey aliens. I worked through problems such as making the monkeys, ‘aliens’ alternatives, move simultaneously, being hit and destroyed with a moving banana.
